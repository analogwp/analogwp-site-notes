A comprehensive solution for agency-client transitions with visual commenting system, task management, and collaborative features.

== Description ==

Site Notes streamlines the process of handing over websites from agencies to clients. It provides a visual commenting system that allows teams to annotate web pages with feedback, manage tasks, and collaborate effectively during the transition period.

= Key Features =

* **Visual Commenting System** - Click on any element to add comments with screenshots
* **Task Management** - Create, assign, and track tasks with priority levels
* **Status Tracking** - Monitor progress with open, in-progress, and resolved statuses
* **Screenshot Capture** - Automatic screenshot generation for visual context
* **Responsive Design** - Works seamlessly on desktop and mobile devices
* **User Permissions** - Proper role-based access control
* **Admin Dashboard** - Comprehensive overview of all comments and tasks
* **Translation Ready** - Full internationalization support

= Perfect For =

* Web agencies streamlining client feedback collection
* Freelance developers managing client collaboration
* Website maintenance teams handling ongoing communication
* Design reviews with visual feedback on layouts
* Bug reporting with clear visual documentation

= How It Works =

1. **Add Comments** - Click on any page element to add a visual comment
2. **Capture Context** - Screenshots are automatically generated for clarity
3. **Set Priorities** - Organize feedback with High, Medium, and Low priorities
4. **Track Progress** - Monitor status from Open to In Progress to Resolved
5. **Collaborate** - Add replies and updates to comments
6. **Manage** - Use the admin dashboard for overview and bulk actions

== Installation ==

= Automatic Installation =

1. Go to Plugins > Add New in your WordPress admin
2. Search for "Site Notes"
3. Click Install and then Activate
4. Navigate to Site Notes in the admin menu

= Manual Installation =

1. Download the plugin ZIP file
2. Go to Plugins > Add New > Upload Plugin
3. Choose the ZIP file and click Install Now
4. Activate the plugin
5. Navigate to Site Notes in the admin menu

== Frequently Asked Questions ==

= Can clients without WordPress accounts add comments? =

Yes, the plugin supports comments from non-logged-in users, making it perfect for client feedback without requiring WordPress accounts.

= Are screenshots automatically generated? =

Yes, when you add a comment by clicking on an element, a screenshot is automatically captured to provide visual context.

= Can I customize the comment statuses? =

Currently, the plugin uses standard statuses: Open, In Progress, and Resolved. Customization options may be added in future versions.

= Is the plugin translation-ready? =

Yes, the plugin is fully translation-ready with proper internationalization support using the 'analogwp-site-notes' text domain.

= What are the minimum requirements? =

WordPress 5.0+, PHP 7.4+, and a modern web browser with JavaScript enabled.

= Does it work with any theme? =

Yes, the plugin is designed to work with any WordPress theme. It uses non-intrusive overlay techniques for the commenting system.
